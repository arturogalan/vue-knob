# vue-knob

> VueJS component inspired by jQuery Knob

## Usage

``` bash
npm install vue-knob

import Vue from 'vue'
import vue-knob from 'vue-knob'
Vue.component(vue-knob)
```
``` html
<vue-knob v-model="myNumber" />
```

Component is SVG and fills the space available.
